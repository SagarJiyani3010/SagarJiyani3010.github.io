    House Price Prediction

    This Project will help you to find out the Best Suitable Selling Price of your House according to 16 different features like Area, Size, Type, Garage, No of Rooms, Quality of the House.

    I have taken the California House Price Dataset from Kaggle. I have removed many features and reduced the Dataset to 16 features so it can be easy for Deployment. I have used PyCaret to train the Model and by using Flask App I have Developed a webpage and Deployed the trained Model.
